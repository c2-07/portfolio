# Introduction An Astro file (.astro) is refreshingly simple. It separates

your server-side logic from your HTML using a code fence (---), much like
Markdown frontmatter.

```astro
---
// Component Script (Server-side logic, purely procedural setup)
// We will define our minimal data here later.
---

<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>gourav@arch</title>
  </head>
  <body class="bg-black text-gray-300 font-mono">
    </body>
</html>
```
